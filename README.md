# Web de boda: Alvaro & Malala

Archivos listos para subir a GitHub Pages.

## Estructura

- `index.html`: pagina principal.
- `css/style.css`: estilos generales de la pagina.
- `css/invitation.css`: animacion inicial del sobre, carta y mariposa.
- `js/main.js`: cuenta regresiva.
- `js/invitation.js`: animacion de entrada.
- `images/`: coloca aqui tus imagenes.

## Imagenes recomendadas

Guarda tus fotos con estos nombres:

- `images/hero.jpg`
- `images/foto1.jpg`
- `images/foto2.jpg`
- `images/foto3.jpg`
- `images/foto4.jpg`

Si no subes esas imagenes, la pagina funcionara, pero apareceran espacios vacios o fondos sin foto.

## Publicacion

Sube todo el contenido de esta carpeta a la raiz del repositorio de GitHub Pages.
